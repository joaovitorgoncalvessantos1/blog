# 📝 Blog Website

## 📖 Sobre o projeto
Este é um projeto de blog desenvolvido com HTML e CSS, com o objetivo de praticar a estruturação de páginas web e organização de conteúdo.

O site simula um blog real, onde os posts são exibidos de forma organizada, permitindo uma boa experiência de leitura para o usuário.

---

## ✨ Funcionalidades
- Exibição de posts
- Estrutura de blog organizada
- Navegação entre seções
- Layout simples e intuitivo

---

## 🎨 Layout
O projeto possui:
- Cabeçalho com identidade do site
- Área principal com posts
- Estrutura semelhante a blogs reais
- Organização clara de conteúdo

---

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3

---

## 🎯 Aprendizados
Durante esse projeto, pratiquei:
- Estruturação de páginas web
- Organização de conteúdo em formato de blog
- Uso de HTML semântico
- Estilização com CSS
